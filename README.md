# cLoki + Clickhouse using S3/MINIO storage

#### BETA VERSION! PLEASE REPORT BUGS AND IMPROVEMENTS

This docker bundle will spin up a cLoki + Clickhouse powered stack w/ Minio/S3 storage

## Components

#### Core
* clickhouse-server
* cLoki
* minio
#### Add-Ons
* grafana
* pastash

## Setup

```bash
docker-compose up
```
